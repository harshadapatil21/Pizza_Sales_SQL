# Pizza_Sales_SQL
Pizza sales analysis project using MySQL. Includes database creation, tables, and SQL queries for insights.

# 🍕 Pizza Sales SQL Project

## 📌 Description
This project is based on pizza sales data analysis using MySQL. 
It includes database creation, tables, and SQL queries to analyze sales performance.

## 🛠️ Tools Used
- MySQL

## 📂 Project Files
- query1.sql -> Database and table creation
- query2.sql -> Highest priced pizza
- query3.sql -> Total number of orders
- query4.sql -> Total revenue calculation
- query5.sql -> Most common pizza size oredred
- query6.sql -> top 5 most orderd pizza types along with their quantities.
- query7.sql -> total quantity of each pizza category ordered
- query8.sql ->  distribution of orders by hour of the day
- query9.sql -> category wise distribution of pizzas.
- query10.sql -> average number of pizzas ordered per day
- query11.sql -> top 3 most oredred pizza type based on revenue
- query12.sql -> percentage contribution of each pizza type to total revenue
- query13.sql -> cumulative revenue generated over time
- query14.sql -> top 3 most ordered pizza types based on revenue for each pizza category

## 📊 Features / Queries
- Create database and tables
- Retrieve total orders
- Find highest priced pizza
- Calculate total revenue

## ▶️ How to Run
1. Open MySQL Workbench
2. Run query1.sql to create database and tables
3. Run other query files step by step
